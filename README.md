# rrolli93.github.io

Personal portfolio of **Rafael Alain Rolli** — biotechnologist & founder building at the intersection of AI and biotech.

🌐 **Live:** https://rrolli93.github.io

## Stack

- [Next.js 14](https://nextjs.org/) — App Router, static export
- TypeScript
- Deployed to **GitHub Pages** via GitHub Actions

## Develop

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # static export → ./out
```

Every push to `main` auto-deploys via [`.github/workflows/deploy.yml`](.github/workflows/deploy.yml).
